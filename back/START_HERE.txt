🎯 מה זה עושה?
---------------

מערכת API מלאה לניהול שיבוצים צבאיים עם:

✅ אלגוריתם שיבוץ חכם - מאזן עומס, שומר על מנוחה, מצב חירום אוטומטי
✅ מערכת הרשאות - 3 רמות: מ"פ, מ"מ, מ"כ
✅ JWT Authentication - אבטחה מלאה
✅ SQLite Database - מסד נתונים מקומי
✅ 35+ API Endpoints - כל מה שצריך
✅ ניהול זמינות - סבבי בית, התש"ב, בקשות יציאה
✅ הסמכות - מעקב אחרי כישורים


🔥 תכונות מיוחדות:
-------------------

1. **הכל דינמי** - אין קוד hard-coded, הכל מוגדר דרך API
2. **אלגוריתם חכם** - מפזר שעות באופן שווה ושומר על מנוחה
3. **מצב חירום אוטומטי** - אם אין מספיק כוח אדם, מקל על הדרישות
4. **אזהרות** - מודיע על מצבים חריגים
5. **תבניות משימות** - ניתן להגדיר כל סוג משימה


📋 מה אני צריך?

- Python 3.8+

- 5 דקות


📖 איך להתחיל?
--------------

1. קרא את README.md - יש שם מדריך מלא
2. הרץ את setup.py - מתקין הכל אוטומטית
3. הרץ את api.py - מפעיל את השרת
4. הרץ את example_usage.py - יוצר מערכת דוגמה מלאה


🚀 צעדים ראשונים:
------------------

בדוק ש-API עובד:
  curl http://localhost:5000/api/health

תירשם:
  curl -X POST http://localhost:5000/api/register \
    -H "Content-Type: application/json" \
    -d '{"username":"commander1","password":"pass123","full_name":"משה כהן"}'

שמור את ה-Token שחוזר!

המשך לפי README.md...


⚠️ חשוב:
--------

1. שמור את ה-Token לאחר Login
2. כל בקשה צריכה: Authorization: Bearer <TOKEN>
3. Token פג תוקף אחרי 7 ימים
4. לפני ייצור - שנה SECRET_KEY ב-.env
